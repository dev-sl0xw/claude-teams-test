# AWS Well-Architected Framework CDK 핸즈온 커리큘럼

## 학습 경로 개요

이 프로젝트는 AWS SAA(Solutions Architect Associate) 수준에서
Well-Architected Framework의 6대 요소를 CDK로 실습하며 학습합니다.

```
[00 기초]  →  [01 운영 우수성]  →  [02 보안]  →  [03 안정성]
                                                      ↓
           [06 지속 가능성]  ←  [05 비용 최적화]  ←  [04 성능 효율성]
```

> 권장 순서대로 학습하되, 기초(00) 완료 후에는 관심 있는 Pillar부터 시작해도 됩니다.

## Pillar 간 연결 관계

각 Pillar는 독립적이지 않습니다. 하나의 결정이 여러 Pillar에 영향을 미칩니다.

| 결정 | 긍정적 영향 | 부정적 영향 |
|------|-----------|-----------|
| Multi-AZ 배포 | 안정성 ↑ | 비용 ↑ |
| 데이터 암호화 | 보안 ↑ | 성능 (미미한) ↓ |
| Auto Scaling | 안정성 ↑, 비용 최적화 ↑ | 운영 복잡성 ↑ |
| 서버리스 전환 | 비용 최적화 ↑, 지속 가능성 ↑ | 성능 (콜드 스타트) ↓ |
| 로깅 활성화 | 운영 우수성 ↑, 보안 ↑ | 비용 ↑ |

> **SAA 시험의 핵심**: 하나의 Pillar만 최적화하는 것이 아니라, **6개 Pillar의 균형**을 찾는 것이 중요합니다.
